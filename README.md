# 💰 Personal Finance Tracker

A simple and responsive Personal Finance Tracker built using **HTML, CSS, and JavaScript**. This application helps users manage their income and expenses, calculate their current balance, and visualize financial data through interactive charts.

## 🚀 Live Demo

👉 https://hasinikalvakk-lang.github.io/personal-finance-tracker/

## 📌 Features

* Add Income and Expense Transactions
* Real-Time Balance Calculation
* Transaction History Management
* Delete Transactions
* Data Persistence using Local Storage
* Interactive Pie Chart Visualization
* Responsive User Interface
* Fully Client-Side Application

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Local Storage API

## ⚙️ How It Works

1. Enter a transaction description.
2. Enter the amount.
3. Select Income or Expense.
4. Click "Add Transaction".
5. The application updates:

   * Total Income
   * Total Expenses
   * Current Balance
   * Expense Chart
6. Data is automatically saved in Local Storage.
